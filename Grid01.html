COL TWO

Using h-100 makes the element to fill up available space. If you want both columns to be the same make sure both are using h-100

Use gutter class g-* to control gutter width/height x,y

<div class="row row-cols-1 row-cols-md-2 gy-2 h-100"><div>

gy-2 sets a space between stacked on Y-axis

You may want to create a minHeight class in site.css, to set a vertical height using vh, eg. min-height: 80vh;

Use p-* for padding, pt-* for top, pb-*etc in the div you are using to wrap content with.

<div class="h-100 bg-secondary text-light p-3"></div>

Set responsive breakpoints to determine how columns will stack on mobile screens.

Use row-cols-1 (smallest breakpoint), row-cols-md-2 to stack columns on breakpoints
